# Multiplayer Setup Instructions

This is a multiplayer-only game where you play against another player.

* The server is not publicly hosted.
* You must play over a LAN or VLAN network.
* If you are using a VLAN, connect to the server using the VLAN IP address.

## Steps

1. Run Server.exe.
2. Copy the IP address displayed in the server console.
3. Run Battleship Game.exe.
4. Enter the server IP address you copied and press Connect.
5. Choose one of the following:

   * Create a new account by filling in the registration fields, then click Register followed by Login.
   * Use a provided template account by simply clicking Login.
6. If you are testing on one computer, launch Battleship Game.exe a second time so you have two game instances running.
7. Log in to the second instance using different account credentials.
8. To start a match:

   * Both players join the same queue, or
   * One player joins Any Queue, or
   * Both players join Any Queue.
9. Once matched, play the game.
